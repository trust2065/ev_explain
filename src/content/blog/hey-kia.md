「Hey Kia」是 Kia 搭載 ccNC 系統（如 EV3）的原生語音助理，旨在讓駕駛在不分心的情況下控制車輛。

以下是它能執行的主要功能與範例指令：

### 1. 車輛舒適控制
*   **空調與溫度**：「Set temperature to 23 degrees」（設定溫度至 23 度）、「I'm cold」（我好冷，系統會自動調高溫度）。
*   **座椅與硬體**：「Turn on steering wheel heater」（開啟方向盤加熱）、「Open all windows」（打開所有車窗）、「Turn on seat ventilation」（開啟座椅通風）。

### 2. 導航與路況
*   **地點搜尋**：「Find nearest charging stations」（尋找最近的充電站）、「Navigate to Sydney Opera House」（導航至悉尼歌劇院）。
*   **行程查詢**：「How much range is left?」（還剩多少續航里程？）、「What is the estimated time of arrival?」（預計抵達時間？）。

### 3. 多媒體與通訊
*   **音樂控制**：「Play FM 104.1」、「Skip this song」（跳過這首歌）。
*   **電話功能**：「Call [Contact Name]」（撥打電話給某人）。

### 4. 生活與天氣資訊
*   **即時查詢**：「What’s the weather in Melbourne today?」（今天墨爾本天氣如何？）。

---

### ⚠️ 針對澳洲版 EV3 的關鍵限制
**語言限制**：澳洲規格的「Hey Kia」原生系統目前僅支援英文指令。它無法辨識中文語音指令。

### 💡 常用指令範例表

| 類別 | 英文指令 (請說...) | 功能說明 (中文) |
| :--- | :--- | :--- |
| **空調與舒適** | Set temperature to 23 degrees | 將空調溫度設定為 23 度 |
| | I'm cold / I'm hot | 系統會自動調升或調降溫度 |
| | Turn on steering wheel heater | 開啟方向盤加熱功能 |
| | Turn on seat ventilation | 開啟座椅通風 (涼爽) |
| **導航與電力** | Find charging stations | 搜尋附近的電動車充電站 |
| | Go home / Go to work | 導航回住家或公司 (需先在系統設定) |
| | How much range is left? | 詢問目前電池還能跑多少公里 |
| | Cancel navigation | 取消目前的導航行程 |
| **硬體控制** | Open all windows | 打開車上所有的車窗 |
| | Open / Close the trunk | 開啟或關閉後車廂 (感應式尾門) |
| | Open the sunroof | 打開天窗 |
| **媒體與通訊** | Play FM 104.1 | 切換至特定的廣播頻率 |
| | Call [聯絡人名稱] | 撥打電話給指定聯絡人 |
| | What time is it? | 詢問目前時間 |

方向盤快捷鍵：

短按語音鍵：喚醒原廠的 Hey Kia（處理車輛硬體，如車窗、空調）。

長按語音鍵：喚醒手機的 Siri 或 Google 助理（處理中文導航、Line 訊息、中文音樂）。
